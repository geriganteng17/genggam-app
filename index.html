Pertanyaan Umum

Datanya disimpan di mana?

Di HP kamu sendiri (penyimpanan lokal). Genggam nggak nyimpen data keuanganmu di server mana pun — jadi privat dan tetap bisa dibuka offline.

Apakah perlu langganan?

Nggak. Sekali bayar Rp99.000, kamu pakai selamanya termasuk update. Nggak ada tagihan bulanan.

Fitur AI-nya bayar lagi?

Nggak. Fitur AI pakai API key Gemini gratis dari Google — tinggal ikutin panduan singkat di dalam app (3 langkah, gratis).

Kalau nggak cocok gimana?

Ada garansi 7 hari. Kalau Genggam nggak sesuai harapan, hubungi kami dalam 7 hari setelah beli — uang kembali, tanpa ribet.

Bisa dipakai di iPhone?

Saat ini Genggam tersedia untuk Android. Versi iOS sedang dipertimbangkan.
